📖 О проекте
Урок 16: Агроном — Точное земледелие — это интерактивный 3D-тренажёр для обучения управлению сельскохозяйственным беспилотником (БПЛА). Симулятор входит в образовательную платформу Anti-AI Qwen Academy, направленную на подготовку специалистов будущего в сфере верификации ИИ-контента и работы с автономными системами.


🎯 Образовательные цели
Освоение принципов точного земледелия с использованием БПЛА
Понимание задач мониторинга сельскохозяйственных полей
Обучение обнаружению проблемных зон (болезни, вредители, засуха)
Развитие навыков пилотирования дрона в условиях ограниченной видимости
Понимание ограничений автономных систем и необходимости человеческого контроля


✨ Возможности

🎮 Геймплей
3 уровня сложности: Лёгкий (3 зоны), Средний (5 зон), Сложный (8 зон)
Реалистичная физика дрона с инерцией и наклонами при манёврах
Система столкновений: любое касание препятствия = провал миссии
Ограниченный ресурс распыляемой жидкости для обработки зон
Таймер миссии с предупреждениями об остатке времени


🚁 Детализированная 3D-модель дрона
Карбоновая рама с титановыми стойками
4 бесколлекторных мотора с золотыми колокольчиками
3-лопастные неоновые пропеллеры с анимацией вращения
Подвесная камера с объективом и системой стабилизации
Бак для жидкости с прозрачной текстурой
RGB-LED индикаторы (зелёный/красный)
Защитные обода вокруг пропеллеров


🌍 Игровой мир
Поле 200×200 метров с сеткой координат
Деревья по периметру (20 шт. с рандомизацией)
Сарай с крышей как ориентир
Столбы-препятствия (5-15 шт. в зависимости от уровня)
Динамическое освещение и тени (PCFSoftShadowMap)
Атмосферный туман для глубины сцены


📊 Телеметрия в реальном времени
Высота полёта (м)
Скорость (м/с)
Уровень заряда батареи (%)
Качество сигнала (%)
NDVI индекс поля
Здоровье растений (%)
Обработано зон / всего зон
Остаток жидкости (%)


🔊 Аудио-система
Фоновая музыка (MP3) с fallback на синтезированную мелодию
Звуковые эффекты через Web Audio API:
Распыление жидкости
Успешная обработка зоны
Столкновение с препятствием
Победа в миссии
